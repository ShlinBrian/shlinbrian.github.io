# How to Publish Your Portfolio (No Coding Required)

Since you want to host this live on the internet and we've named your folder `shlinbrian.github.io`, the absolute best and easiest way to host this for free is using **GitHub Pages**. 

Here are the exact, step-by-step instructions to get your website live.

## Step 1: Create a GitHub Account
1. Go to [github.com](https://github.com) in your web browser.
2. Click **Sign up** in the top right corner.
3. Follow the prompts to create an account. (If you already have a GitHub account with the username `shlinbrian`, you can skip this step!)

## Step 2: Create a New Repository
*A "repository" is just a storage space for your website files.*
1. Once logged in to GitHub, look for the **"+"** icon in the top right corner of the page and click **New repository**.
2. **Repository name:** You *must* name it exactly like this: `[your-username].github.io` 
   *(For example, if your username is `shlinbrian`, name it exactly `shlinbrian.github.io`)*
3. **Description:** (Optional) "My personal portfolio."
4. **Public/Private:** Make sure **Public** is selected so people can see your website.
5. Skip the other checkboxes (like adding a README).
6. Click the green **Create repository** button.

## Step 3: Upload Your Files
1. You will be taken to a page with a lot of code commands. Ignore them!
2. Look for the link that says **"uploading an existing file"** (it’s a small link near the center of the page) and click it.
3. Open the `shlinbrian.github.io` folder on your Mac (the one we just worked in).
4. Select all the files inside this folder:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `material` (the folder)
5. Drag and drop all of them into the box on the GitHub website. Wait for all files to finish uploading.
6. Scroll down to the **"Commit changes"** button at the bottom and click it. (This just means "save my files".)

## Step 4: Turn on GitHub Pages (If it doesn't happen automatically)
Sometimes GitHub does this automatically for repositories ending in `.github.io`, but let's double-check:
1. On your repository page in GitHub, click the **Settings** tab near the top right.
2. In the left sidebar, click on **Pages**.
3. Under "Build and deployment", ensure the **Source** is set to "Deploy from a branch".
4. Under the "Branch" section, select **`main`** (or `master`) from the dropdown menu, keep the folder as `/ (root)`, and click **Save**.

## Step 5: Wait and View Your Website
- GitHub needs about 1-3 minutes to officially launch your site. 
- Open a new tab and type in your URL: `https://shlinbrian.github.io`
- Your modern, creative agency portfolio is now live on the internet! 

## Making Future Updates
If you ever want to change a photo or text:
1. Edit the file on your Mac.
2. Go to your repository on GitHub.
3. Click "Add file" > "Upload files".
4. Drag and drop the modified file, and click "Commit changes". It will update live automatically after a minute.
